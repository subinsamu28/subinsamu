# Assets Directory

This folder contains static assets for the portfolio website.

## Required Files

Please place the following files in this directory:

1. **favicon-optimized.png** - Browser tab icon (recommended size: 32x32 or 64x64 pixels)
2. **logo-final.png** - Social media sharing image (recommended size: 1200x630 pixels for optimal Open Graph display)

## File Locations

These files are referenced in:
- `/index.html` - for favicon and Open Graph meta tags
- Social media platforms will use `logo-final.png` when your portfolio is shared

## Notes

- Make sure files are optimized for web (compressed PNG format)
- The logo should represent your personal brand
- The favicon will appear in browser tabs and bookmarks
